Functionality of adding, retreiving and deleting a mapping validation test suite. No update is possible.<br/><br/>Each validation test suite is identified and versioned. Minimal metadata are associated. <br/><br/>Each validation test suite is placed in a broader scheme for transformation rule governance documented elsewhere. <br/><br/>If the validation test suite is already used in any notice validation, i.e. an RDF manifestation exists, then that rule set cannot be deleted. Only marked for deprecation.<br/><br/>
