A set of constraints over notice metadata.  <br/><br/>Normally this would be encoded in a simple predicate logic, but certainly less expressive models may be sufficient. <br/>For example a constraint can be formulated over the "form type" metadata field. It can say that form number can be either "F03, F06, F26": "form number = [F03, F06, F26]". <br/>Default logical condition over the array of possible values, is OR. <br/>We can imagine that the "legal basis" needs to be specified in addition to the form type. For example: "legal basis=D24" . The logical condition between fields is AND.<br/><br/>
